# Snake game

The old and famous Snake game created by yourself!
This project is optimal for beginners, even if the code may appear a bit difficult initially.

===============================

Components :

- 1 Arduino Pro Mini
- 4 Matrix Led 8x8
- 1 Analogic Joystick
